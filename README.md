# PHP ИНН API — Получение реквизитов компании по ИНН

> **Автор README:** Claude (Anthropic)

## Описание

PHP-скрипт для получения актуальных реквизитов юридического лица или индивидуального предпринимателя по ИНН через API сервиса [api-fns.ru](https://api-fns.ru). Поддерживает получение **исторических данных** — реквизитов компании на конкретную дату (адрес и руководитель могут меняться со временем).

---

## Возможности

- Поиск по ИНН юридического лица (10 цифр) или ИП (12 цифр)
- Получение текущих реквизитов компании
- Получение реквизитов **на указанную дату** (адрес, руководитель)
- Поддержка нескольких компаний в одном ответе (несколько записей по ИНН)

---

## Требования

- PHP 8.1+
- Расширение `curl`
- API-ключ от [api-fns.ru](https://api-fns.ru)

---

## Установка

1. Скопируйте файл `main.php` на ваш сервер.
2. Вставьте ваш API-ключ в константу:

```php
define('API_FNS_KEY', 'ВАШ_КЛЮЧ_ЗДЕСЬ');
```

---

## Использование

### GET-запрос

```
GET /main.php?inn=7707083893&date=01.01.2022
```

### POST-запрос (JSON)

```http
POST /main.php
Content-Type: application/json

{
  "inn": "7707083893",
  "date": "01.01.2022"
}
```

### POST-запрос (form-data)

```
POST /main.php
inn=7707083893&date=01.01.2022
```

---

## Параметры

| Параметр | Тип    | Обязательный | Описание                                              |
|----------|--------|:------------:|-------------------------------------------------------|
| `inn`    | string | ✅           | ИНН компании (10 цифр) или ИП (12 цифр)               |
| `date`   | string | ❌           | Дата в формате `дд.мм.гггг`. По умолчанию — сегодня   |

---

```json

  "ИНН": "7707083893",
  "ОГРН": "1027700132195",
```

---

## Коды ответов

| HTTP-код | Описание                                |
|----------|-----------------------------------------|
| `200`    | Успех                                   |
| `400`    | Неверный запрос (отсутствует ИНН и т.д.)|
| `502`    | Ошибка при обращении к api-fns.ru        |

---

## Архитектура

| Функция             | Описание                                                                 |
|---------------------|--------------------------------------------------------------------------|
| `fetchFromFns()`    | Выполняет запрос к api-fns.ru и передаёт данные на парсинг               |
| `parse()`           | Определяет тип записи (ЮЛ или ИП) и вызывает нужный парсер              |
| `parseUL()`         | Парсит данные юридического лица                                          |
| `parseIP()`         | Парсит данные индивидуального предпринимателя                            |
| `findHistoryInfo()` | Ищет в истории реквизиты, актуальные на дату `REQUEST_DATE`              |
| `getEndData()`      | Формирует итоговый ответ, приоритизируя исторические данные              |
| `curlGet()`         | Универсальная обёртка для HTTP GET-запросов через cURL                   |
